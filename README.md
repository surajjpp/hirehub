## HireHub

HireHub is a simple job portal made with **React**, **Node.js**, **Express** and **MongoDB**.

Students can create an account, update their profile and apply for jobs.  
Recruiters can create a company, post jobs and see applicants.

### Tech stack
- React + Vite (frontend)
- Tailwind CSS
- Redux Toolkit
- Node.js + Express (backend)
- MongoDB with Mongoose

### How to run backend
1. Open the `backend` folder.
2. Install packages: `npm install`
3. Create a `.env` file with your MongoDB URL, JWT secret and Cloudinary keys.
4. Start server: `npm run dev`

### How to run frontend
1. Open the `frontend` folder.
2. Install packages: `npm install`
3. Start app: `npm run dev`
4. Open the shown localhost URL in the browser.

This project is for learning full stack development.
